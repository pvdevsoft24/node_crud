To start the server :

node index.js

To stop the server :

ctrl + c

To start Mongo DB 
sudo systemctl start mongod

Verify that MongoDB has started successfully.
sudo systemctl status mongod

Stop MongoDB
sudo systemctl stop mongod

Restart MongoDB.
sudo systemctl restart mongod
